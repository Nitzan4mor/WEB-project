Favorite People

This web application fetch 12 randoms persons from Api each time and allows you to do the following:
  1. Display each person basic details - name, image and age
  2. Display more details - email, phone and cell
  3. Add person to your favorite list using local storage
  4. Display all your favorite persons
  5. Remove a single person from the favorist list
  6. Remove all of the favorit persons from the list

Enjoy!
